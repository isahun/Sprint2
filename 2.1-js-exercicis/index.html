2.1 JavaScript Tests

Exercici 1.1: Arrow Functions

Nivell 1

Exercici 1: Conversió de funcions. Tens una funció add que accepta dos paràmetres i retorna la seva suma. Converteix-la en una funció de fletxa. Exemple: function add(a, b) { return a + b; }

Exercici 2: Funció de fletxa sense paràmetres. Crea una funció de fletxa anomenada randomNumber que no necessiti paràmetres i que retorni un número aleatori entre 0 i 100.

Exercici 3: Ús de this en les funcions de fletxa. Crea una classe Person que tingui una propietat name i una funció greet que utilitzi una funció de fletxa. La funció ha d'imprimir una salutació amb el nom.

Nivell 2

Exercici 4: Funció de fletxa dins d'un loop. Crea una funció printNumbers que accepti un array de números i utilitzi un loop for per imprimir cada número a la consola utilitzant una funció de fletxa.

Nivell 3

Exercici 5: Funció de fletxa amb setTimeout. Crea una funció de fletxa que imprimeixi un missatge a la consola després d'esperar 3 segons.

Exercici 1.2: Operador ternari

Nivell 1

Exercici 1: Operador ternari bàsic. Escriu una funció potConduir que accepti l'edat com a paràmetre i utilitzi l'operador ternari per determinar si l'usuari pot conduir.

Exercici 2: Ús amb operadors de comparació. Escriu una expressió amb operador ternari que determini quin dels dos nombres (num1 i num2) és més gran.

Nivell 2

Exercici 3: Ús enllaçat d'operadors ternaris. Escriu una expressió que determini si un número és positiu, negatiu o zero.

Exercici 4: Operador ternari amb funcions. Crea una funció trobarMaxim que accepti tres paràmetres i utilitzi l'operador ternari per trobar el valor màxim.

Nivell 3

Exercici 5: Operador ternari dins un bucle. Escriu una funció parOImpar que accepti un array de números i determini si cada número és parell o imparell.

Exercici 1.3: Callbacks

Nivell 1

Exercici 1: Callback bàsic. Escriu una funció processar que accepti un nombre i una funció de callback. La funció ha d'invocar el callback passant el nombre com a paràmetre.

Exercici 2: Callbacks amb operacions matemàtiques. Escriu una funció calculadora que accepti dos nombres i una funció de callback. La funció ha d'invocar el callback amb els dos nombres com a paràmetres. Després, crida calculadora amb una funció que faci la suma.

Nivell 2

Exercici 3: Ús de callbacks en funcions asíncrones. Escriu una funció esperarISaludar que accepti un nom i una funció de callback. La funció ha d'esperar 2 segons i després invocar el callback passant el nom.

Exercici 4: Callbacks amb arrays. Escriu una funció processarElements que accepti un array i una funció de callback. La funció ha d'invocar el callback per cada element de l'array.

Nivell 3

Exercici 5: Callbacks amb cadenes. Escriu una funció processarCadena que accepti una cadena de text i una funció de callback. La funció ha de convertir la cadena a majúscules i invocar el callback amb la cadena transformada.

Exercici 1.4: Rest & Spread Operators

Nivell 1

Exercici 1: Operador Spread en arrays. Crea dues arrays (array1 i array2) i utilitza l'operador spread per crear una tercera array que contingui tots els elements.

Exercici 2: Operador Rest en funcions. Crea una funció suma que utilitzi l'operador rest per acceptar un nombre indeterminat d'arguments i retornar la seva suma.

Nivell 2

Exercici 3: Copiant objectes amb Spread. Crea un objecte objecte1 i després crea objecte2 com una còpia utilitzant spread. Modifica una propietat de objecte2 i comprova que objecte1 no canvia.

Exercici 4: Rest en destructuring. Crea una array amb diversos elements i utilitza destructuring amb l'operador rest per assignar els dos primers elements a variables i la resta a una tercera variable.

Nivell 3

Exercici 5: Spread en funcions. Crea una funció que accepti tres arguments. Després crea una array amb tres elements i crida la funció utilitzant spread.

Exercici 6: Fusionant objectes amb Spread. Crea dos objectes amb propietats diferents i utilitza spread per fusionar-los en un de nou.

Exercici 1.5: Array transformations

Nivell 1

Exercici 1: map. Donat l'array [1, 2, 3, 4], crea una nova array amb el quadrat de cada número.

Exercici 2: filter. Donat l'array [1, 2, 3, 4], crea una nova array només amb els números parells.

Exercici 3: find. Donat l'array [1, 10, 8, 11], utilitza find per trobar el primer número major que 10.

Exercici 4: reduce. Donat l'array [13, 7, 8, 21], calcula la suma total utilitzant reduce.

Nivell 2

Exercici 5: Donat l'array [1, 3, 7, 10, 15, 17, 11, 5, 8, 12, 9], crea una funció en una sola línia que: filtri els nombres ≥ 10, els multipliqui per 2 i calculi la suma total.

Nivell 3

Exercici 6: every / some. Utilitza every i some per comprovar si tots o alguns dels elements de l'array [11, 12, 13, 14] són majors que 10.

Exercici 2: for-of. Utilitza un bucle for-of per imprimir cada nom de l'array.

Exercici 3: filter. Donada l'array [1, 2, 3, 4, 5, 6], crea una nova array amb només els parells.

Exercici 1.7: Promises & Async/Await

Nivell 1

Exercici 1: Creació d'una promesa. Crea una promesa que es resolgui després de 2 segons i retorni el text 'Hola, món'.

Exercici 2: Utilització d'una promesa. Utilitza .then per imprimir el resultat a la consola.

Exercici 3: Promesa amb reject. Crea una promesa que es resolgui si l'input és 'Hola' i es rebutgi en qualsevol altre cas.

Exercici 4: async / await. Escriu una funció asíncrona que esperi la promesa i imprimeixi el resultat.

Nivell 3
